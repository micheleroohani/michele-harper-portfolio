# Michele Harper portfolio

A modern single-page UX portfolio website built with Vite, React, Tailwind, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize before launch

- Replace `hello@example.com` in `src/main.jsx`
- Replace the LinkedIn URL in `src/main.jsx`
- Add a resume PDF and link to it
- Add approved project images or screenshots if available

## Deploy

This can be deployed to GitHub Pages, Netlify, or Vercel.
